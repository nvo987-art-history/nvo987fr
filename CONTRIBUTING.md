# Contributing

Thank you for your interest in contributing to the NVO987 project.

## About

NVO987 – Culture Visuelle Moderne et Contemporaine  
https://nvo987.fr

This repository contains the public website of the NVO987 cultural and research association.

## Scope

The website is a static site focused on:

- visual culture research
- contemporary artistic practices
- cultural studies
- academic publications

## Contributions

At this stage the repository is primarily maintained by the project founder.

External contributions may include:

- corrections
- documentation improvements
- metadata improvements
- accessibility suggestions
